# indolib_rails

Usage :

How to use my plugin.
IndolibRails.hi("**timezone**")

Installation :

Add this line to your application's Gemfile:
$ gem 'indolib_rails', '~> 1.3'

And then execute:
$ bundle

Or install it yourself as:
$ gem install indolib_rails
